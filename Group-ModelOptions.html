Group - Model Options

Model::AFT_s_s$

To specify the model for the AFt ettringite - tricarboalu03 phases:

  • s_s_nonideal : Use a non-ideal solid-solution model for ettringite and tricarboalu03
  • s_s_ideal : Use an ideal solid-solution model for ettringite and tricarboalu03
  • s_s_no : Include ettringite and tricarboalu03 as pure phase

This model option is only applicable for the CEMDATA thermodynamic database.

Cement::Model::MSH$

To specify the model for the MSH phases M075SH and M15SH:

  • s_s_ideal : Use an ideal solid-solution model 
  • s_s_no: Include the phases as pure phases

This model option is only applicable for the CEMDATA thermodynamic database.

Cement::Model::Monosulfoaluminate

No effect (to be checked)

Cement::Model::Zeolites::Na

If checked, Na-zeolites are included in the cement model

Cement::Model::Zeolites::K

If checked, K-zeolites are included in the cement model

Cement::Model::Zeolites::Ca

If checked, Ca-Zeolites are included in the cement model.


After execution, the exact model definition can be checked via the file CementModel1._in in the simulation folder.
