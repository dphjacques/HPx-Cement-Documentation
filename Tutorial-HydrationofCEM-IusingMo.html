Tutorial - Hydration of CEM-I using Modified Parrot-Killoch equations

Tutorial -  Hydration of CEM-I using Modified Parrot-Killoch equations


Description

Simulate the hydration of CEM-I (100% OPC clinkers) of 100 g OPC clinkers with a w/c ratio of 0.5 at 25°C for 1000 days.


You will learn to

Use the template to simulate hydration of OPC CEM-I with the modified Parrot-Killoch parameters


Template - Hydration_10_CEM-I


Depends on

Tutorial - Create Managed Project with Cement Module


Step 1 – Create the simulation from the template to calculate hydration of OPC clinkers

Go to Project Manager

Select the project in the bottom panel

Go to Templates tab in the top panel

Select the template “Hydration_10_CEM-I”

Click the “Create new from template” button

The Metadata dialogue window opens.

Define the name of the simulation (Hydration CEM-I)

Click OK

The Project Manager is active again. 

Select the simulation (Hydration CEM-I) and click Select

Step 2 – Define the hydration process 

Go to Global Definitions

Go to Global Variables

Go to Cement Model tab

Select system 17 (Ca-Si-Al-Fe-S-Mg-C-Na-K-Cl) from record Cement::Model::System$

Select s_s_ideal for Model::AFT_s_s$

Cement recipe uses the default values

Go to Hydration tab

Change Cement::Hydration::Time to 1000

Run

Step 4 – Check Log and Plots

Go to the Graph Workspace

Click the Disabling Automatic Graph Update icon (icon change to )


Click the Update the plot icon
