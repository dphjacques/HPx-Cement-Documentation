  1. Introduction

    Glossary

    Case

    When a simulation contains an input table and parameter values are defined in that table, a case represents one particular calculation in the simulation with that set of parameters. Calculation results can be saved in a separate subfolder in the simulation folder.

    See Tutorial - Calculate the cement hydrates assemblages with part of OPC is replaced by GBSF

    Cement Systems

    The oxides (e.g., CaO, SiO2, ...) that are taken into account in the cement models

    See Available cement systems

    Managed Project

    a Managed Project is a project with a specific structure for input data, geochemical simulations and reactive transport calculations together with auxiliary information. Beside a tool for traceability and reproducibility, it also facilitates exchange of the project with other users. It also integrates the project in HPGeochemistry with the project group in HYDRUS.

    See Managed Projects in online HPx manual.

    Parametric study

    Based on a global input table, a series of calculations will be performed in which selected numerical values will vary automatically based on (i) start or end position of a parameter vector, (ii) parameter vectors listed in a table, (iii) centered around a parameter vector, or (iv) gridded parameter vectors.
